# RAG Knowledge Base System

A lightweight, modular RAG (Retrieval-Augmented Generation) pipeline in pure Python.

## Overview

This library provides a clean abstraction for each stage of a RAG pipeline:

- **Document Processing** — chunk text with configurable size and overlap
- **Embedding Generation** — generate vector embeddings (OpenAI or local models)
- **Vector Storage** — store and retrieve vectors (FAISS or Chroma)
- **RAG Pipeline** — glue class for retrieval-augmented QA
