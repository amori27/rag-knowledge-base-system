# Security Policy

## Supported Versions

We currently support the latest release with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue.

Instead, send a private report to **amori27** via GitHub's security advisory feature:

https://github.com/amori27/rag-knowledge-base-system/security/advisories

We will respond within 72 hours and work with you to resolve the issue promptly.

## Responsible Disclosure

We ask that you:

- Give us reasonable time to fix the issue before disclosing it publicly.
- Provide a clear description of the vulnerability and steps to reproduce.
- Avoid exploiting the vulnerability or accessing data beyond what's necessary to demonstrate the issue.
